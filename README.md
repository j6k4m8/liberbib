# liberbib

An open-source bibliography-management tool.

Intended to fuel projects like [remarkabib](/https://github.com/j6k4m8/remarkabib) and an [upcoming] web app.

## Usage

```bash
uv run liberbib search --bibtex "ten simple rules"
```
